# Abdullahi Ibrahim - Premium Professional Portfolio

A high-end, premium web portfolio designed for businesses, startups, and NGOs. This project showcases the intersection of software development and human behavior (Sociology & Psychology).

## 🚀 Built With
- **React**: Frontend framework.
- **Tailwind CSS**: Modern styling and layout.
- **Framer Motion**: Smooth animations and transitions.
- **Lucide React**: Premium icon set.
- **Vite**: Ultra-fast build tool.

## ✨ Key Features
- **Mandatory Welcome Overlay**: A professional "Splash Screen" entry point that focuses user attention and locks scrolling until a selection is made.
- **100% Responsive Design**: Fully optimized for phones, tablets, and desktops.
- **High-Conversion Contact Flow**: Integrated WhatsApp redirect for instant client communication.
- **Psychology-Driven UX**: Designed with user behavior in mind, leveraging a background in Sociology and Psychology.
- **Dynamic Portfolio**: Showcases real-world projects with live links and technology badges.
- **Interactive Journey Timeline**: Visualizes professional and academic growth.
- **Tech Arsenal**: A curated display of modern technologies used in development.
- **Performance Optimized**: Built with modern best practices for fast load times and smooth interactions.

## 🛠️ Project Structure
- `src/components/WelcomeOverlay.jsx`: The mandatory entry gate.
- `src/components/Hero.jsx`: High-impact landing section.
- `src/components/About.jsx`: Detailed professional background.
- `src/components/Portfolio.jsx`: Project showcase with glassmorphism.
- `src/components/Contact.jsx`: State-managed form with WhatsApp integration.
- `src/App.jsx`: Main application logic and scroll management.

## ⚡ Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
